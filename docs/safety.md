# Safety and Privacy

## Do Not Publish

- Real Obsidian vaults
- Unpublished manuscripts
- Raw or processed clinical data
- Patient-level data
- Real project paths with personal names
- API keys, tokens, provider configs
- Private reviewer comments
- Real statistical outputs unless intentionally released

## Evidence Status

AI output is `Unverified until checked`.

The MCP server writes working memory, not final scientific truth.

## Allowed Automatic Writeback

- Task summaries
- Materials read
- File changes
- Pending verification items
- Evidence gaps
- Figure version rows

## Requires User Confirmation

- Final conclusions
- Statistical interpretations
- Medical conclusions
- Literature evidence grades
- Manuscript-ready claims
- Formal project recommendations

## Recommended Git Hygiene

Use `.gitignore` for:

```text
.env
*.key
*.pem
data/
raw/
private/
*.xlsx
*.csv
*.parquet
*.docx
*.pdf
```

Only commit fake examples or explicitly public data.
